## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { Alert } from '@/vibes/soul/primitives/alert';

function Usage() {
    return <Alert message="This is a success alert" variant="success" />;
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### AlertProps

| name                                                                                    | type                                          | default     |
| --------------------------------------------------------------------------------------- | --------------------------------------------- | ----------- |
| `variant*` <Tooltip content='The style variant of the alert, defines color and icon' /> | `"success" \| "warning" \| "error" \| "info"` |             |
| `message*` <Tooltip content='Alert message displayed' />                                | `ReactNode`                                   |             |
| `description`                                                                           | `string`                                      |             |
| `dismissLabel`                                                                          | `string`                                      | `"Dismiss"` |
| `action`                                                                                | `{ label: string; onClick: () => void; }`     |             |
| `onDismiss`                                                                             | `() => void`                                  |             |
