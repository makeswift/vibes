## Usage

<CodeBlock lang="bash">{`import { Alert } from '@/vibes/soul/primitives/alert'
import { useState } from 'react'`}</CodeBlock>

<CodeBlock lang="bash">{`const [showAlert, setShowAlert] = useState(false)`}</CodeBlock>

<CodeBlock lang="bash">{`<Alert variant="success" message="1 item added to your cart" showAlert={showAlert} onClose={setShowAlert}/>`}</CodeBlock>

## API Reference

# Props

| name                                                                                    | type                                                   | optional | default |
| --------------------------------------------------------------------------------------- | ------------------------------------------------------ | -------- | ------- |
| `variant*` <Tooltip content='The style variant of the alert, defines color and icon' /> | `"success" \| "warning" \| "error" \| "info"`          | `false`  |         |
| `message*` <Tooltip content='Alert message displayed' />                                | `ReactNode`                                            | `false`  |         |
| `description`                                                                           | `string \| undefined`                                  | `true`   |         |
| `dismissLabel`                                                                          | `string \| undefined`                                  | `true`   |         |
| `action`                                                                                | `{ label: string; onClick: () => void; } \| undefined` | `true`   |         |
| `onDismiss`                                                                             | `(() => void) \| undefined`                            | `true`   |         |
