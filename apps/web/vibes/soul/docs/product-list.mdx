## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { ProductList } from '@/vibes/soul/sections/product-list';

function Usage() {
    return (
        <ProductList products={products} />
    );
}

const products = [
    {
      id: '1',
      title: 'Product 1',
      href: '#',
    },
]
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### ProductList

| Prop                 | Type                                                                                              | Default                                           |
| -------------------- | ------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| `products*`          | [`Streamable`](/docs/soul/streamable)`<ListProduct[]>` <Tooltip content="See ListProduct below"/> |                                                   |
| `compareProducts`    | [`Streamable`](/docs/soul/streamable)`<ListProduct[]>` <Tooltip content="See ListProduct below"/> | `[]`                                              |
| `className`          | `string`                                                                                          |                                                   |
| `colorScheme`        | `'light' \| 'dark'`                                                                               | `'light'`                                         |
| `aspectRatio`        | `'5:6' \| '3:4'  \| '1:1'`                                                                        | `'5:6'`                                           |
| `showCompare`        | `boolean`                                                                                         | `false`                                           |
| `compareAction`      | `(formData: FormData) => void`                                                                    |                                                   |
| `compareLabel`       | [`Streamable`](/docs/soul/streamable)`<string>`                                                   | `'Compare'`                                       |
| `compareParamName`   | `string`                                                                                          | `'compare'`                                       |
| `emptyStateTitle`    | [`Streamable`](/docs/soul/streamable)`<string>`                                                   | `'No products found'`                             |
| `emptyStateSubtitle` | [`Streamable`](/docs/soul/streamable)`<string>`                                                   | `'Try browsing our complete catalog of products'` |
| `placeholderCount`   | `number`                                                                                          | `0`                                               |
