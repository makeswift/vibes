## Usage

Import the component.

<CodeBlock lang="bash">{`import { ProductList } from '@/vibes/soul/primitives/product-list'`}</CodeBlock>

Query the product data.

<CodeBlock lang="bash">{`const products = await getProducts()`}</CodeBlock>

Display the component.

<CodeBlock lang="bash">{`<ProductList products={products} />`}</CodeBlock>

### Streaming

The `products` property accepts either `Product[]` or `Promise<Product[]>`. By passing a `Promise<Product[]>`, you enable streaming which will show a loading state until the data is finished loading and streamed to the browser.
