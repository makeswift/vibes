## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
'use client';

import { SwatchRadioGroup } from '@/vibes/soul/form/swatch-radio-group';

function Usage() {
    return (
        <SwatchRadioGroup
          options={[
            { type: 'color', label: 'Option 1', value: 'option-1', color: 'red' },
            { type: 'color', label: 'Option 2', value: 'option-2', color: 'green' },
            { type: 'color', label: 'Option 3', value: 'option-3', color: 'blue' },
          ]}
        />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### SwatchRadioGroupProps

| Prop                 | Type                      | Default   |
| -------------------- | ------------------------- | --------- |
| `className`          | `string`                  |           |
| `label`              | `string`                  |           |
| `options`            | `SwatchOption[]`          |           |
| `errors`             | `string[]`                |           |
| `colorScheme`        | `'light' \| 'dark'`       | `'light'` |
| `onOptionMouseEnter` | `(value: string) => void` |           |

### SwatchOption

```ts
type SwatchOption =
  | {
      type: 'color';
      value: string;
      label: string;
      color: string;
      disabled?: boolean;
    }
  | {
      type: 'image';
      value: string;
      label: string;
      image: { src: string; alt: string };
      disabled?: boolean;
    };
```
