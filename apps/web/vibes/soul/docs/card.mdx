## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { Card } from '@/vibes/soul/primitives/card';

function Usage() {
    return (
      <Card href="#" title="Partial shade" image={{ src: 'https://rstr.in/monogram/vibes/RopDQNbjTc_', alt: 'Low Maintenance' }} />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### CardProps

| Prop              | Type                           | Default   |
| ----------------- | ------------------------------ | --------- |
| `className`       | `string`                       |           |
| `title*`          | `string`                       |           |
| `image`           | `{ src: string; alt: string }` |           |
| `href*`           | `string`                       |           |
| `textColorScheme` | `'light' \| 'dark'`            | `'light'` |
| `iconColorScheme` | `'light' \| 'dark'`            | `'light'` |
| `aspectRatio`     | `5:6' \| '3:4' \| '1:1`        | `'5:6'`   |

### CSS Variables

This component supports various CSS variables for theming. Here's a comprehensive list, along
with their default values:

```css
:root {
  --card-focus: hsl(var(--primary));
  --card-light-text: hsl(var(--foreground));
  --card-light-icon: hsl(var(--foreground));
  --card-light-background: hsl(var(--contrast-100));
  --card-dark-text: hsl(var(--background));
  --card-dark-icon: hsl(var(--background));
  --card-dark-background: hsl(var(--contrast-500));
  --card-font-family: var(--font-family-body);
}
```
