## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { Card } from '@/vibes/soul/primitives/card';

function Usage() {
    return (
        <Card>
            <h3>Card Title</h3>
            <p>Card content goes here</p>
        </Card>
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### CardProps

| Prop        | Type                                                                                    | Default |
| ----------- | --------------------------------------------------------------------------------------- | ------- |
| `as`        | `ElementType`                                                                           | `'div'` |
| `className` | `string`                                                                                |         |
| `children*` | `ReactNode`                                                                             |         |
| `href`      | `string`                                                                                |         |
| `ariaLabel` | `string` <Tooltip content="Required when href is provided for accessibility purposes"/> |         |

### CardSkeletonProps

| Prop        | Type     | Default |
| ----------- | -------- | ------- |
| `className` | `string` |         |
