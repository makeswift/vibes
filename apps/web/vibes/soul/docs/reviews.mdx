## API Reference

### ReviewsProps

| Prop                | Type                                                                                                              | Default     |
| ------------------- | ----------------------------------------------------------------------------------------------------------------- | ----------- |
| `reviews*`          | [`Streamable`](/docs/soul/streamable)`<Review[]>` <Tooltip content="See Review below"/>                           |             |
| `averageRating*`    | [`Streamable`](/docs/soul/streamable)`<number>`                                                                   |             |
| `totalCount`        | [`Streamable`](/docs/soul/streamable)`<number>`                                                                   |             |
| `paginationInfo`    | [`Streamable`](/docs/soul/streamable)`<CursorPaginationInfo>` <Tooltip content="See CursorPaginationInfo below"/> |             |
| `nextLabel`         | [`Streamable`](/docs/soul/streamable)`<string>`                                                                   |             |
| `previousLabel`     | [`Streamable`](/docs/soul/streamable)`<string>`                                                                   |             |
| `emptyStateMessage` | `string`                                                                                                          |             |
| `reviewsLabel`      | `string`                                                                                                          | `'Reviews'` |

### Review

```css
}
```
