## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { ProductCard } from '@/vibes/soul/primitives/product-card';

function Usage() {
    return (
      <ProductCard
        product={{
          id: '1',
          href: '#',
          title: 'Product Name',
        }}
      />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

You will need to nest the `ProductCard` component(s) within a `CompareDrawerProvider` if you set the `showCompare` prop to `true`.

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { CompareDrawerProvider } from '@/vibes/soul/primitives/compare-drawer';
import { ProductCard } from '@/vibes/soul/primitives/product-card';

function Usage() {
    return (
      <CompareDrawerProvider items={[]}>
      <ProductCard
        product={{
          id: '1',
          href: '#',
          title: 'Product Name',
        }}
        showCompare
      />
      </CompareDrawerProvider>
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### ProductCardProps

| Prop               | Type                                             | Default                                                                                                       |
| ------------------ | ------------------------------------------------ | ------------------------------------------------------------------------------------------------------------- |
| `className`        | `string`                                         |                                                                                                               |
| `colorScheme`      | `'light' \| 'dark'`                              | `'light'`                                                                                                     |
| `aspectRatio`      | `'5:6' \| '3:4' \| '1:1'`                        | `'5:6'`                                                                                                       |
| `showCompare`      | `boolean`                                        | `false'`                                                                                                      |
| `imagePriority`    | `boolean`                                        | `false`                                                                                                       |
| `imageSizes`       | `string`                                         | `'(min-width: 80rem) 20vw, (min-width: 64rem) 25vw, (min-width: 42rem) 33vw, (min-width: 24rem) 50vw, 100vw'` |
| `compareLabel`     | `string`                                         |                                                                                                               |
| `compareParamName` | `string`                                         |                                                                                                               |
| `borderRadius`     | `'none' \| 'small' \| 'medium' \| 'large'`       | `'large'`                                                                                                     |
| `product*`         | `Product` <Tooltip content="See Product below"/> |                                                                                                               |

### Product

| Prop       | Type                           | Default |
| ---------- | ------------------------------ | ------- |
| `id*`      | `string`                       |         |
| `title*`   | `string`                       |         |
| `href*`    | `string`                       |         |
| `image`    | `{ src: string; alt: string }` |         |
| `price`    | `Price`                        |         |
| `subtitle` | `string`                       |         |
| `badge`    | `string`                       |         |
| `rating`   | `number`                       |         |
