## API Reference

### SlideshowProps

| Prop         | Type                                           | Default |
| ------------ | ---------------------------------------------- | ------- |
| `className`  | `string`                                       |         |
| `slides*`    | `Slide[]` <Tooltip content="See Slide below"/> |         |
| `playOnInit` | `boolean`                                      | `true`  |
| `interval`   | `number`                                       | `5000`  |

### Slide

| Prop              | Type                                         | Default |
| ----------------- | -------------------------------------------- | ------- |
| `title*`          | `string`                                     |         |
| `description`     | `string`                                     |         |
| `showDescription` | `boolean`                                    |         |
| `image`           | `Image` <Tooltip content="See Image below"/> |         |
| `cta`             | `CTA` <Tooltip content="See CTA below"/>     |         |
| `showCta`         | `boolean`                                    |         |

### Image

| Prop          | Type     | Default |
| ------------- | -------- | ------- |
| `alt*`        | `string` |         |
| `blurDataUrl` | `string` |         |
| `src*`        | `string` |         |

### CTA

| Prop      | Type                         | Default |
| --------- | ---------------------------- | ------- |
| `label*`  | `string`                     |         |
| `href*`   | `string`                     |         |
| `variant` | `ButtonLinkProps['variant']` |         |
| `size`    | `ButtonLinkProps['size']`    |         |
| `shape`   | `ButtonLinkProps['shape']`   |         |

See [ButtonLink](/docs//soul/button-link) for more details on the `variant`, `size`, and `shape` properties.
