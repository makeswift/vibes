## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { Tabs } from '@/vibes/soul/primitives/tabs';

function Usage() {
    return (
        <Tabs
          defaultValue="tracking"
          tabs={[
            { value: 'tracking', label: 'Tracking', content: <p>Tracking</p> },
            { value: 'details', label: 'Details', content: <p>Details</p> },
          ]}
        />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### TabsProps

| Prop            | Type                                       | Default |
| --------------- | ------------------------------------------ | ------- |
| `className`     | `string`                                   |         |
| `defaultValue*` | `string`                                   |         |
| `tabs*`         | `Tab[]` <Tooltip content="See Tab below"/> |         |

### Tab

| Prop       | Type        | Default |
| ---------- | ----------- | ------- |
| `value*`   | `string`    |         |
| `label*`   | `string`    |         |
| `content*` | `ReactNode` |         |
