## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { Footer } from '@/vibes/soul/sections/footer';

function Usage() {
    return (
        <Footer
            contactInformation={contactInformation}
            logo={logo}
            sections={sections}
        />
    );
}
export const sections = [
    {
      title: 'Help & Support',
      links: [
        { label: 'FAQs', href: '#' },
        { label: 'Contact Us', href: '#' },
        { label: 'Returns', href: '#' },
        { label: 'Shipping', href: '#' },
      ],
    },
];

export const logo = 'SOUL';

export const contactInformation = {
    address: 'info@soul.com',
    phone: '+(1) 408 123 4567',
};
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### FooterProps

| Prop                 | Type                                                                                                          | Default |
| -------------------- | ------------------------------------------------------------------------------------------------------------- | ------- |
| `logo*`              | [`Streamable`](/docs/soul/streamable)`<string \| Image>` <Tooltip content="See Image below"/>                 |         |
| `sections*`          | [`Streamable`](/docs/soul/streamable)`<Section[]>` <Tooltip content="See Section below"/>                     |         |
| `copyright`          | [`Streamable`](/docs/soul/streamable)`<string>`                                                               |         |
| `contactInformation` | [`Streamable`](/docs/soul/streamable)`<ContactInformation>` <Tooltip content="See ContactInformation below"/> |         |
| `paymentIcons`       | [`Streamable`](/docs/soul/streamable)`<ReactNode[]>`                                                          |         |
| `socialMediaLinks`   | [`Streamable`](/docs/soul/streamable)`<SocialMediaLink[]>` <Tooltip content="See SocialMediaLink below"/>     |         |
| `contactTitle`       | `string`                                                                                                      |         |
| `className`          | `string`                                                                                                      |         |
| `logoHref`           | `string`                                                                                                      |         |
| `logoLabel`          | `string`                                                                                                      |         |
| `logoWidth`          | `number`                                                                                                      |         |
| `logoHeight`         | `number`                                                                                                      |         |

### Section

| Prop     | Type                                         | Default |
| -------- | -------------------------------------------- | ------- |
| `title`  | `string`                                     |         |
| `links*` | `Link[]` <Tooltip content="See Link below"/> |         |

### Link

| Prop     | Type     | Default |
| -------- | -------- | ------- |
| `label*` | `string` |         |
| `href*`  | `string` |         |

### ContactInformation

| Prop      | Type     | Default |

### SocialMediaLink

| Prop    | Type        | Default |

### Image

| ------ | -------- | ------- |
