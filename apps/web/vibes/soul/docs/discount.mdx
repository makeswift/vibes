## Usage

<CodeBlock lang="ts">{`

import { Discount } from '@/vibes/soul/sections/discount';

function Usage() {

    return (
        <Discount
          backgroundImage="https://storage.googleapis.com/s.mkswft.com/RmlsZTpiMjMxMjAyZS05MDFjLTRjY2YtYWNiYS02ZmY1ZWQxZTVlZGQ=/electric-discount.jpeg"

## API Reference

### DiscountProps

| Prop               | Type                                                         | Default |
| ------------------ | ------------------------------------------------------------ | ------- |
| `id*`              | `string`                                                     |         |
| `backgroundImage*` | `string`                                                     |         |
| `discounts*`       | `DiscountType[]` <Tooltip content="See DiscountType below"/> |         |
| `onDismiss`        | `() => void`                                                 |         |

### DiscountType

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `label` | `string` |         |
| `code`  | `string` |         |
