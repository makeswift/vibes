## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
'use client';

import { Discount } from '@/vibes/soul/sections/discount';

function Usage() {

    return (
        <Discount
          backgroundImage="https://rstr.in/monogram/vibes/-ZKYHmBgOcN"
          discounts={[
            {
              label: '20% off',
              code: 'TAKE20',
            },
            {
              label: '5% off',
              code: 'TAKE5',
            },
          ]}
          id="example-discount"
        />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### DiscountProps

| Prop               | Type                                                         | Default |
| ------------------ | ------------------------------------------------------------ | ------- |
| `id*`              | `string`                                                     |         |
| `backgroundImage*` | `string`                                                     |         |
| `discounts*`       | `DiscountType[]` <Tooltip content="See DiscountType below"/> |         |
| `onDismiss`        | `() => void`                                                 |         |

### DiscountType

| Prop    | Type     | Default |
| ------- | -------- | ------- |
| `label` | `string` |         |
| `code`  | `string` |         |
