## API Reference

### TextareaProps

| Prop          | Type                | Default   |
| ------------- | ------------------- | --------- |
| `className`   | `string`            |           |
| `prepend`     | `React.ReactNode`   |           |
| `label`       | `string`            |           |
| `errors`      | `string[]`          |           |
| `colorScheme` | `'light' \| 'dark'` | `'light'` |
