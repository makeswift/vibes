---
title: Cart
description: The Cart component displays the items in the user's cart, along with a summary of the total cost.
preview:
  Electric: cart-electric
  Warm: cart-warm
  Luxury: cart-luxury
previewSize: xl
features:
  - Displays the items in the user's cart
  - Displays a summary of the total cost
  - Allows the user to remove items from the cart
  - Allows the user to update the quantity of items in the cart
---

## API Reference
