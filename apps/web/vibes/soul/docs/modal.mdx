## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
'use client';

import { useState } from 'react';

import { Modal } from '@/vibes/soul/primitives/modal';

function Usage() {
    const [isOpen, setIsOpen] = useState(true);

    return (
      <Modal
        isOpen={isOpen}
        setOpen={setIsOpen}
        title="Example modal"
        trigger={<button>Open Modal</button>}
      >
        <p>Put your content here!</p>
      </Modal>
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

This component uses the Dialog component from Radix UI. Refer to the [Radix UI Dialog documentation](https://www.radix-ui.com/primitives/docs/components/dialog) for more information.

### ModalProps

| Prop         | Type                      | Default |
| ------------ | ------------------------- | ------- |
| `className`  | `string`                  |         |
| `isOpen`     | `boolean`                 |         |
| `setOpen`    | `(open: boolean) => void` |         |
| `required`   | `boolean`                 | `false` |
| `hideHeader` | `boolean`                 | `false` |
| `title*`     | `string`                  |         |
| `trigger*`   | `ReactNode`               |         |
| `children*`  | `ReactNode`               |         |
