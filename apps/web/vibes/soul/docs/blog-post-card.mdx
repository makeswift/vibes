## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { BlogPostCard } from '@/vibes/soul/primitives/blog-post-card';

function Usage() {
    return (
      <BlogPostCard
        title: 'Vestibulum eleifend placerat ligula and even more text for a long title'
        author="Ryan Smith"
        content="'Lorem ipsum dolor sit amet, consectetur adipiscing elit. Quisque eget velit hendrerit erat imperdiet tincidunt. Lorem ipsum dolor sit amet, consectetur adipiscing elit. This is some more text to test out here."
        date="2022-01-01"
        href="#"
        image: {{
          src: 'https://storage.googleapis.com/s.mkswft.com/RmlsZTo1YzIwNTljMi04NzcwLTRiM2ItYmIzMy02ZTk0ODNkY2M5MDk=/mini-bar-bag.jpeg',
          alt: 'Vestibulum eleifend placerat ligula',
        }},
        href="/blog/how-to-create-a-blog-post-card"
      />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### BlogPostCardProps

| Prop        | Type                                    | Default |
| ----------- | --------------------------------------- | ------- |
| `className` | `string`                                |         |
| `title*`    | `string`                                |         |
| `author`    | `string \| null`                        |         |
| `content*`  | `string`                                |         |
| `date*`     | `string`                                |         |
| `image`     | `{ src: string; alt: string; } \| null` |         |
| `href*`     | `string`                                |         |
