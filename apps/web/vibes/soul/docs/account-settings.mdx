## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { AccountSettingsSection } from '@/vibes/soul/sections/account-settings';

function Usage() {
    return (
        <AccountSettingsSection
            account={{ firstName: '', lastName: '', email: '' }}
            changePasswordAction={changePasswordAction}
            updateAccountAction={updateAccountAction}
        />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### AccountSettingsProps

| Prop                        | Type                                                                                | Default              |
| --------------------------- | ----------------------------------------------------------------------------------- | -------------------- |
| `title`                     | `string`                                                                            | `'Account Settings'` |
| `account*`                  | `Account`                                                                           |                      |
| `updateAccountAction*`      | `UpdateAccountAction` <Tooltip content="See update-account-form.tsx for detail"/>   |                      |
| `updateAccountSubmitLabel`  | `string`                                                                            |                      |
| `changePasswordTitle`       | `string`                                                                            | `'Change Password'`  |
| `changePasswordAction*`     | `ChangePasswordAction` <Tooltip content="See change-password-form.tsx for detail"/> |                      |
| `changePasswordSubmitLabel` | `string`                                                                            |                      |
