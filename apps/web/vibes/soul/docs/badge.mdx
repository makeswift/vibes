## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { Badge } from '@/vibes/soul/primitives/badge';

function Usage() {
    return <Badge>New</Badge>;
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### BadgeProps

| Prop                                                      | Type                                                       | Default     |
| --------------------------------------------------------- | ---------------------------------------------------------- | ----------- |
| `className`                                               | `string`                                                   |             |
| `children*` <Tooltip content="The text to be displayed"/> | `string`                                                   |             |
| `shape`                                                   | `'pill' \| 'rounded'`                                      | `"rounded"` |
| `variant`                                                 | `'primary' \| 'warning' \| 'error' \| 'success' \| 'info'` | `"primary"` |
