## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { Navigation } from '@/vibes/soul/primitives/navigation';

const navigationLinks = [
    {
      label: 'Shop All',
      href: '#',
    }
]

function Usage() {
    return (
      <Navigation
          accountHref="#"
          cartHref="#"
          links={navigationLinks}
          logo="SOUL"
          searchHref="#"
      />
    );
}
`}
</CodeBlock>

#### Actions

You can find the type defintions for the actions below.

```ts
type Action<State, Payload> = (
  state: Awaited<State>,
  payload: Awaited<Payload>,
) => State | Promise<State>;

type LocaleAction = Action<SubmissionResult | null, FormData>;
type CurrencyAction = Action<SubmissionResult | null, FormData>;

type SearchAction<S extends SearchResult> = Action<
  {
    searchResults: S[] | null;
    lastResult: SubmissionResult | null;
    emptyStateTitle?: string;
    emptyStateSubtitle?: string;
  },
  FormData
>;
```

This component uses Confom to handle form submissions. Refer to the [Conform docs](https://conform.guide/) for more details.
