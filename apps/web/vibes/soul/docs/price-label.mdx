## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { PriceLabel } from '@/vibes/soul/primitives/price-label';

function Usage() {
    return (
        <PriceLabel price="100" />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### PriceLabelProps

| Prop          | Type                                | Default   |
| ------------- | ----------------------------------- | --------- |
| `className`   | `string`                            |           |
| `colorScheme` | `'light' \| 'dark'`                 | `'light'` |
| `price*`      | `string \| PriceRange \| PriceSale` |           |

### PriceRange

| Prop        | Type      | Default |
| ----------- | --------- | ------- |
| `type*`     | `'range'` |         |
| `minValue*` | `string`  |         |
| `maxValue*` | `string`  |         |

### PriceSale

| Prop             | Type     | Default |
| ---------------- | -------- | ------- |
| `type*`          | `'sale'` |         |
| `previousValue*` | `string` |         |
| `currentValue*`  | `string` |         |
