## API Reference

### CalendarProps

| Prop          | Type                | Default   |
| ------------- | ------------------- | --------- |
| `children*`   | `ReactNode`         |           |
| `className`   | `string`            |           |
| `colorScheme` | `'light' \| 'dark'` | `'light'` |

Refer to the [React Day Picker](https://daypicker.dev/) documentation for additional properties.
