## API Reference

### CompareDrawerProps

| Prop          | Type                                                                   | Default     |
| ------------- | ---------------------------------------------------------------------- | ----------- |
| `items*`      | `CompareDrawerItem[]` <Tooltip content="See CompareDrawerItem below"/> |             |
| `paramName`   | `string`                                                               | `'compare'` |
| `action`      | `ComponentProps<'form'>['action']`                                     |             |
| `submitLabel` | `string`                                                               | `'Compare'` |

### CompareDrawerItem

| Prop     | Type                           | Default |
| -------- | ------------------------------ | ------- |
| `id*`    | `string`                       |         |
| `image?` | `{ src: string; alt: string }` |         |
| `href*`  | `string`                       |         |
| `title*` | `string`                       |         |
