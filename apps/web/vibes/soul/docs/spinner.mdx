## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { Spinner } from '@/vibes/soul/primitives/spinner';

function Usage() {
    return (
        <Spinner size="lg" />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### SpinnerProps

| Prop               | Type                           | Default        |
| ------------------ | ------------------------------ | -------------- |
| `size`             | `'xs' \| 'sm' \| 'md' \| 'lg'` | `'sm'`         |
| `loadingAriaLabel` | `string`                       | `'Loading...'` |

### CSS Variables

This component supports various CSS variables for theming. Here's a comprehensive list.

<CodeBlock lang="css">{`
:root {
    --spinner-base: var(--contrast-100);
    --spinner-ring: var(--primary-shadow);
}
`}</CodeBlock>
