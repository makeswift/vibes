## Usage

<CodeBlock lang="ts">{`
import { OffsetPagination } from '@/vibes/soul/primitives/offset-pagination';

function Usage() {
    return (
        <OffsetPagination pages={8} />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

This component maintains state for the current page by using a [search parameter](https://nextjs.org/docs/app/api-reference/functions/use-search-params) called `page`. While this component will update the search param with the current page, it is your responsibility to handle changes to the selected page.

Refer to the Learn Next.js tutorial section on [adding search and pagination](https://nextjs.org/learn/dashboard-app/adding-search-and-pagination) for more background on how to handle that.

### OffsetPaginationProps

| Prop     | Type     | Default |
| -------- | -------- | ------- |
| `pages*` | `number` |         |
