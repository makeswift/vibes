## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
'use client';

import { RadioGroup } from '@/vibes/soul/form/radio-group';

function Usage() {
    return (
        <RadioGroup
          options={[
            { value: 'option-1', label: ' Option 1' },
            { value: 'option-2', label: ' Option 2' },
            { value: 'option-3', label: ' Option 3' },
          ]}
        />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

This component uses the Radix UI RadioGroup component. Refer to the [RadioGroup documentation](https://www.radix-ui.com/primitives/docs/components/radio-group) for more information.

### RadioGroupProps

| Prop                 | Type                                             | Default   |
| -------------------- | ------------------------------------------------ | --------- |
| `className`          | `string`                                         |           |
| `label`              | `string`                                         |           |
| `options*`           | `Option[]` <Tooltip content="See Option below"/> |           |
| `errors`             | `string[]`                                       |           |
| `colorScheme`        | `'light' \| 'dark'`                              | `'light'` |
| `onOptionMouseEnter` | `(value: string) => void`                        |           |

### Option

| Prop       | Type      | Default |
| ---------- | --------- | ------- |
| `value*`   | `string`  |         |
| `label*`   | `string`  |         |
| `disabled` | `boolean` |         |
