## Usage

<CodeBlock lang="ts">{`
import { Input } from '@/vibes/soul/form/input';

function Usage() {
    return (
      <Input label="username" />
    );
}
`}
</CodeBlock>

## API Reference

### InputProps

| Prop          | Type                | Default   |
| ------------- | ------------------- | --------- |
| `className`   | `string`            |           |
| `prepend`     | `React.ReactNode`   |           |
| `label`       | `string`            |           |
| `hideLabel`   | `boolean`           | `true`    |
| `errors`      | `string[]`          |           |
| `colorScheme` | `'light' \| 'dark'` | `'light'` |
