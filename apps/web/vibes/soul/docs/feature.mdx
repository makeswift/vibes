## API Reference

### FeatureProps

| Prop           | Type                                         | Default |
| -------------- | -------------------------------------------- | ------- |
| `image*`       | `Image` <Tooltip content="See Image below"/> |         |
| `title*`       | `string`                                     |         |
| `description*` | `string`                                     |         |
| `cta*`         | `Link` <Tooltip content="See Link below"/>   |         |

### Image

| Prop   | Type     | Default |
| ------ | -------- | ------- |
| `src*` | `string` |         |
| `alt*` | `string` |         |

### Link

| Prop     | Type     | Default |
| -------- | -------- | ------- |
| `href*`  | `string` |         |
| `label*` | `string` |         |

### CSS Variables

  --feature-font-family: var(--font-family-body);
