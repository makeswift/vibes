## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { BlogPost, BlogPostContent } from '@/vibes/soul/sections/blog-post-content';

function Usage() {
    return <BlogPostContent blogPost={blogPost} />;
}

const blogPost = {
    author: 'Sam Smith',
    content: 'My Blog Post',
    date: 'June 30, 2024',
    image: {
        src: 'https://storage.googleapis.com/s.mkswft.com/RmlsZTo1NGFjNmRiYi0xZDE2LTRiOTEtOWUyZS0zMjY1ZjBmZTk0ZjU=/plant-blog-1.jpeg',
        alt: 'A plant with large leaves and a short stem.',
    },
    title: "Top 5 Plants to Purify Your Home's Air",
};
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### BlogPostContentProps

| Prop          | Type                                                                                            | Default |
| ------------- | ----------------------------------------------------------------------------------------------- | ------- |
| `className`   | `string`                                                                                        |         |
| `blogPost*`   | [`Streamable`](/docs/soul/streamable)`<BlogPost>` <Tooltip content="See BlogPost below"/>       |         |
| `breadcrumbs` | [`Streamable`](/docs/soul/streamable)`<Breadcrumb[]>` <Tooltip content="See Breadcrumb below"/> |         |
| `className`   | `string`                                                                                        |         |

### BlogPost

| Prop       | Type                                   | Default |
| ---------- | -------------------------------------- | ------- |
| `title*`   | `string`                               |         |
| `author`   | `string`                               |         |
| `date*`    | `string`                               |         |
| `tags`     | `Tag[] \| null`                        |         |
| `content*` | `string`                               |         |
| `image`    | `{ src: string, alt: string } \| null` |         |

### Breadcrumb

| Prop     | Type     | Default |
| -------- | -------- | ------- |
| `label*` | `string` |         |
| `href*`  | `string` |         |

### Tag

| Prop     | Type                               | Default |
| -------- | ---------------------------------- | ------- |
| `label*` | `string`                           |         |
| `link*`  | ` { href: string, target string }` |         |
