## API Reference

### FeatureGridProps

| Prop     | Type                                                 | Default |
| -------- | ---------------------------------------------------- | ------- |
| `image*` | `Image` <Tooltip content="See Image below"/>         |         |
| `grid*`  | `GridItem[]` <Tooltip content="See GridItem below"/> |         |
| `cta*`   | `Link` <Tooltip content="See Link below"/>           |         |

### GridItem

| Prop           | Type                                                                              | Default |
| -------------- | --------------------------------------------------------------------------------- | ------- |
| `icon*`        | `IconName` <Tooltip content="See the Lucide docs for a full list of icon names"/> |         |
| `title*`       | `string`                                                                          |         |
| `description*` | `string`                                                                          |         |

Refer to the [Lucide docs](https://lucide.dev/icons/) for a full list of available icons.

### Image

| Prop   | Type     | Default |
| ------ | -------- | ------- |
| `src*` | `string` |         |
| `alt*` | `string` |         |

### Link

| Prop     | Type     | Default |
| -------- | -------- | ------- |
| `href*`  | `string` |         |
| `label*` | `string` |         |

### CSS Variables

This component supports various CSS variables for theming. Here's a comprehensive list.

```css
:root {
  --feature-grid-background: color-mix(in oklab, hsl(var(--primary)), black 75%);
  --feature-grid-image-background: hsl(var(--primary) / 10%);
  --feature-grid-font-family: var(--font-family-body);
  --feature-grid-icon: var(--background);
  --feature-grid-title: hsl(var(--background));
  --feature-grid-description: hsl(var(--contrast-200));
}
```
