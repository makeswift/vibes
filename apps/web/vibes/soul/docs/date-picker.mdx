## API Reference

### DatePickerProps

| Prop           | Type                                | Default   |
| -------------- | ----------------------------------- | --------- |
| `defaultValue` | `string \| Date`                    |           |
| `disabledDays` | `CalendarProps['disabled']`         |           |
| `errors`       | `string[]`                          |           |
| `onSelect`     | `(date: Date \| undefined) => void` |           |
| `selected`     | `Date \| undefined`                 |           |
| `colorScheme`  | `'light' \| 'dark'`                 | `'light'` |
