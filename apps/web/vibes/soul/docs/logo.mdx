## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { Logo } from '@/vibes/soul/primitives/logo';

function Usage() {
    return (
      <Logo height={40} href="#" label="SOUL" logo="SOUL" width={200} />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

### LogoProps

| Prop        | Type                                                                                    | Default |
| ----------- | --------------------------------------------------------------------------------------- | ------- |
| `className` | `string`                                                                                |         |
| `logo*`     | [`Streamable`](/docs/soul/streamable)`<string \| { src: string; alt: string } \| null>` |         |
| `label*`    | `string`                                                                                |         |
| `href*`     | `string`                                                                                |         |
| `width*`    | `number`                                                                                |         |
| `height*`   | `number`                                                                                |         |
