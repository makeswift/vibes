## Usage

{/* prettier-ignore-start */}

<CodeBlock lang="ts">{`
import { ToggleGroup } from '@/vibes/soul/form/toggle-group';

function Usage() {
    return (
      <ToggleGroup
        label="Options"
        options={[
          { value: 'option-1', label: ' Option 1' },
          { value: 'option-2', label: ' Option 2' },
          { value: 'option-3', label: ' Option 3' },
        ]}
        type="multiple"
      />
    );
}
`}
</CodeBlock>

{/* prettier-ignore-end */}

## API Reference

This component uses the Toggle Group component from Radix UI. Refer to the [Radix Toggle Group documentation](https://www.radix-ui.com/primitives/docs/components/toggle-group) for more information.

### ToggleGroupProps

| Prop          | Type                                             | Default   |
| ------------- | ------------------------------------------------ | --------- |
| `label`       | `string`                                         |           |
| `options*`    | `Option[]` <Tooltip content="See Option below"/> |           |
| `errors`      | `string[] \| null`                               | `null`    |
| `colorScheme` | `'light' \| 'dark'`                              | `'light'` |
| `hideLabel`   | `boolean`                                        | `false`   |

### Option

| Prop       | Type      | Default |
| ---------- | --------- | ------- |
| `value*`   | `string`  |         |
| `label*`   | `string`  |         |
| `disabled` | `boolean` |         |
