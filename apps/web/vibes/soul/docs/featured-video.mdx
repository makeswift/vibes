## Usage

<CodeBlock lang="ts">{`
import { FeaturedVideo } from '@/vibes/soul/sections/featured-video';

function Usage() {
    return (
        <FeaturedVideo

## API Reference

### FeaturedVideoProps

| Prop           | Type                                       | Default  |
| -------------- | ------------------------------------------ | -------- |
| `title*`       | `string`                                   |          |
| `description*` | `string`                                   |          |
| `video*`       | `string`                                   |          |
| `cta*`         | `Link` <Tooltip content="See Link below"/> |          |
| `mediaAlign`   | `'left' \| 'right' \| 'full'`              | `'left'` |

### Link

| Prop     | Type     | Default |
| -------- | -------- | ------- |
| `href*`  | `string` |         |
| `label*` | `string` |         |

### CSS Variables

This component supports various CSS variables for theming. Here's a comprehensive list.

```css
:root {
  --featured-video-background: color-mix(in oklab, hsl(var(--primary)), black 75%);

  --featured-video-font-family: var(--font-family-body);
  --featured-video-title: hsl(var(--background));
  --featured-video-description: hsl(var(--background));
}
```
