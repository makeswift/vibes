## API Reference

### FeaturedVideoProps

| Prop           | Type                                       | Default  |
| -------------- | ------------------------------------------ | -------- |
| `title*`       | `string`                                   |          |
| `description*` | `string`                                   |          |
| `video*`       | `string`                                   |          |
| `cta*`         | `Link` <Tooltip content="See Link below"/> |          |
| `mediaAlign`   | `'left' \| 'right' \| 'full'`              | `'left'` |

### Link

| Prop     | Type     | Default |
| -------- | -------- | ------- |
| `href*`  | `string` |         |
| `label*` | `string` |         |
