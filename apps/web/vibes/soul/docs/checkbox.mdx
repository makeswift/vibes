## API Reference

This component uses primitives from Radix UI. Refer to [Radix Checkbox](https://www.radix-ui.com/primitives/docs/components/checkbox) and [Radix Label](https://www.radix-ui.com/primitives/docs/components/checkbox) for more information.

### CheckboxProps

| Prop          | Type                | Default   |
| ------------- | ------------------- | --------- |
| `className`   | `string`            |           |
| `label`       | `ReactNode`         |           |
| `errors`      | `string[]`          | `'light'` |
| `colorScheme` | `'light' \| 'dark'` | `'light'` |
