## API Reference

### CounterProps

| Prop                 | Type     | Default            |
| -------------------- | -------- | ------------------ |
| `className`          | `string` |                    |
| `current`            | `number` | `0`                |
| `max`                | `number` | `10`               |
| `decrementAriaLabel` | `string` | `'Decrease count'` |
| `incrementAriaLabel` | `string` | `'Increase count'` |
